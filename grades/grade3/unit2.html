第二单元：万以内的加法和减法（一）

口算两位数加减两位数

1学习目标

📌 本课重点

  • 会口算两位数加两位数
  • 会口算两位数减两位数
  • 能解决实际问题
2口算加法

📝 方法一：先加整十数，再加个位

  • 例：45 + 23
  • 45 + 20 = 65，65 + 3 = 68

📝 方法二：分别加整十和个位

  • 例：45 + 23
  • 40 + 20 = 60，5 + 3 = 8，60 + 8 = 68
3口算减法

📝 方法

  • 例：65 - 32
  • 60 - 30 = 30，5 - 2 = 3，30 + 3 = 33

📝 退位减法

  • 例：72 - 38
  • 72 - 30 = 42，42 - 8 = 34
