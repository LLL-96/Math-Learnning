2认识圆

圆的各部分

圆心(O)：圆中心的点
半径(r)：从圆心到圆上任意一点的线段
直径(d)：通过圆心并且两端都在圆上的线段

直径与半径的关系：d = 2r 或 r = d/2

3圆的周长和面积
周长公式
C = 2πr 或 C = πd
面积公式
S = πr²

圆周率

圆周率 π ≈ 3.14
圆的周长与直径的比值是一个固定的数，叫做圆周率。
