第六单元：分数的加法和减法

同分母、异分母分数加减

1学习目标

📌 本课重点

  • 掌握同分母分数加减法
  • 掌握异分母分数加减法
  • 能进行分数混合运算
2同分母分数加减

📝 方法

  • 分母不变，分子相加减
  • 例：2/7 + 3/7 = (2+3)/7 = 5/7
  • 例：5/9 - 2/9 = (5-2)/9 = 3/9 = 1/3
3异分母分数加减

📐 方法

  • 先通分，再按同分母分数加减法计算
  • 例：1/2 + 1/3 = 3/6 + 2/6 = 5/6
