# 🧮 L-MATH 乐学数学

小学数学互动练习平台，基于人教版教材编写。

🔗 **在线访问**：https://lll-96.github.io/Math-Learnning/practice/index.html

## 功能特点

- 📚 **紧扣教材** — 1-6年级上下册，共88个单元
- 🎮 **互动练习** — 选择、填空、排序、匹配、拖拽多种题型
- 📱 **响应式设计** — 支持手机、平板、电脑
- 🖨️ **打印功能** — 可生成纸质练习题

## 技术栈

纯前端实现：HTML + CSS + JavaScript，无需后端服务器

## 本地运行

```bash
git clone https://github.com/LLL-96/Math-Learnning.git
cd Math-Learnning
# 直接用浏览器打开 index.html 即可
```

## 项目结构

```
├── index.html          # 首页
├── practice/           # 练习中心
│   ├── index.html      # 单元列表
│   ├── quiz.html       # 练习题页面
│   └── practice.js     # 题库数据
└── grades/             # 各年级内容页
```
